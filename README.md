# Portfolio Personal - Sección Hero

Esta es la sección principal (Hero) de mi portfolio personal, construida con **React**, **CSS moderno** y **Lucide Icons**.

## 🚀 Características
- **Diseño Responsivo:** Adaptado para móviles, tablets y pantallas de escritorio usando CSS Grid y `clamp()`.
- **Efectos Premium:** Fondo con malla de gradiente (Mesh Grid) y halos de luz dinámicos (Glow effects).
- **Accesibilidad:** Estructura semántica y optimización de carga de imágenes (LCP).

## 🛠️ Tecnologías utilizadas
- React (Vite)
- Componentes de iconos: `lucide-react`
- CSS3 puro (Variables y Animaciones)
